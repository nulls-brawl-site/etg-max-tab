# ETG MAX Tab v2

Release layout for MAX Tab `2.0.0`.

The bridge dex is built from the latest tested MAX Tab v2 source and is loaded by
`plugin/etg_max.py`.

## Files

- `plugin/etg_max.py` - ExteraGram loader plugin.
- `build/etg-max-bridge.dex` - compiled Java bridge dex.
- `build/etg-max-bridge.dex.sha256` - dex checksum.
- `dex/src/com/etgmax/bridge/MaxBridge.java` - build source.
- `sources/com/etgmax/bridge/MaxBridge.java` - published source copy.
